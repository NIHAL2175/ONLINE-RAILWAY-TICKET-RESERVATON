# 🚆 ONLINE RAILWAY TICKET RESERVATION

## 📌 Project Description

Online Railway Ticket Reservation is a web application that allows users to search for trains, check seat availability and book railway tickets online. The system provides a simple and efficient way to manage passenger details and confirm reservations through a user-friendly interface.

## 🚀 Features

* Search trains by source and destination
* Check seat availability
* Book railway tickets online
* Enter and manage passenger details
* Simple and easy-to-use interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Database (MySQL / SQLite depending on implementation)

## ⚙️ Installation

1. Download or clone the repository:

```
git clone https://github.com/your-username/online-railway-ticket-reservation.git
```

2. Open the project folder.

3. Run the project by opening the `index.html` file in your browser.

## ▶️ Usage

1. Enter source and destination stations.
2. Search for available trains.
3. Select a train and enter passenger details.
4. Confirm the booking to reserve the ticket.
